# Glossary

| Term | Meaning |
| --- | --- |
| CAEPF | Cadastro de Atividade Economica da Pessoa Fisica, used for some employer registrations. |
| CBO | Classificacao Brasileira de Ocupacoes, worker occupation code. |
| CNAE | Classificacao Nacional de Atividades Economicas, establishment activity code. |
| CNPJ | Brazilian legal-entity tax identifier. |
| CPF | Brazilian individual tax identifier. |
| Competence | Payroll/accounting period, normally `YYYY-MM`. |
| DLQ | Dead-letter queue for terminal or exhausted messages. |
| DTO | Typed data transfer object sent by SGP; not XML. |
| eSocial | Brazilian government digital labor, payroll, tax, and social-security reporting system. |
| Event family | Regulatory event class such as S-1299 or S-2200. |
| Exclusion | Regulatory cancellation/removal event, commonly S-3000. |
| FGTS | Fundo de Garantia do Tempo de Servico. |
| Idempotency key | Deterministic versioned key that prevents duplicate regulatory submissions. |
| Leiaute | Official eSocial schema/layout version, such as S-1.3. |
| Rectification | Correcting a previously accepted event while preserving traceability. |
| RGPS | Regime Geral de Previdencia Social. |
| RPPS | Regime Proprio de Previdencia Social. |
| SOAP stub | Deterministic local SOAP response harness used in CI and local tests. |
| Spool | SGP-facing update stream carrying status transitions and payload summaries. |
| Totalizer | S-50xx return event carrying calculated tax/social-security totals. |
| XSD | XML Schema Definition used to validate generated eSocial XML. |
